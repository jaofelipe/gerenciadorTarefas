# Gerenciador de Tarefas

## Como configurar e rodar o projeto

Entrar na pasta do projeto e executar: 
```bash
docker-compose -up --build
```

Para parar rodar o comando: 
```bash
docker-compose down
```

#### Ambientes:
Local: http://localhost:5000

Docker: http://localhost:8080
